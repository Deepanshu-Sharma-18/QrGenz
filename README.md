## About QrGenerator
A QrGenerator app using compose and Hilt based on modern Android tech-stacks. Data is processed and converted into QRcode using zxing library.
## Made Using

* Jetpack Compose
* kotlin
* zxing library (Qr code)
* Hilt and dagger
